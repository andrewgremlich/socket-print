Review code changes, pull requests, or recently modified files for quality, correctness, and adherence to project patterns.

## When to Invoke

Automatically invoke when:
- User asks to review changes or a PR
- Significant refactoring has been completed
- Before creating a commit or PR (if user requests)

## Action

Delegate to the **code-reviewer** agent for a structured review. See `.claude/agents/code-reviewer.md` for the detailed checklist.
